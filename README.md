# Jenkins Pipeline Template

Jenkins Declarative Pipeline (Jenkinsfile) template for CI/CD.

## Features
- CI: Lint, Test, Build
- CD: Docker Build & Push
- Ready for Kubernetes / Helm
- Branch-based deployment
- Clean & reusable structure

## Usage
1. Copy `Jenkinsfile` into your project root
2. Update build steps based on your tech stack
3. Configure Jenkins credentials
4. Create a Jenkins Pipeline job pointing to your repo
